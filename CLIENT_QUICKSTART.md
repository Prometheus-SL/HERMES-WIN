# HERMES WIN - Instalacion rapida

## Que archivo tengo que abrir

Usa `HERMES-WIN-Setup-<version>.exe`.

El `.zip` incluido en la release es una copia portable para soporte tecnico o despliegues manuales, pero el instalador es la opcion recomendada para usuarios finales.

## Pasos para el usuario final

1. Ejecuta `HERMES-WIN-Setup-<version>.exe`.
2. Acepta la instalacion y, si Windows lo pide, concede permisos de administrador.
3. Abre `HERMES WIN`.
4. Inicia sesion con las credenciales facilitadas por tu equipo de soporte.
5. Comprueba que el estado del agente aparece como conectado.
6. Pulsa `Instalar servicio` para que el agente siga funcionando aunque cierres la aplicacion.
7. Si quieres confirmarlo, reinicia el equipo y vuelve a abrir la aplicacion para revisar el estado.

## Si algo falla

- Si la instalacion del servicio pide permisos, acepta la elevacion de Windows.
- Si el agente no conecta, revisa la URL del servidor y vuelve a iniciar sesion.
- Si soporte te pide evidencias, abre el panel de logs de HERMES y comparte el contenido reciente.
